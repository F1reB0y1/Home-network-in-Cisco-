Network Lab Project (Cisco Packet Tracer)
📌 Overview

This project demonstrates the design and configuration of a small enterprise network using VLANs, inter-VLAN routing, and DHCP.

🛠 Technologies Used
Cisco Packet Tracer
VLAN (802.1Q)
DHCP
Router-on-a-Stick
🧱 Network Topology
1 Router
2 Switches
Access Point (Wi-Fi)
4 PCs + 1 Laptop
🌐 Network Design
VLAN 10 — Office (192.168.10.0/24)
VLAN 20 — Guest (192.168.20.0/24)
⚙️ Configuration
Configured VLANs on switches
Assigned access ports to VLANs
Configured trunk links between switches and router
Implemented inter-VLAN routing (Router-on-a-Stick)
Set up DHCP pools for automatic IP assignment
🔍 Troubleshooting
Fixed DHCP issues (APIPA problem)
Resolved native VLAN mismatch
Verified trunk configuration
Tested connectivity using ping and ipconfig
✅ Results
Devices successfully receive IP addresses via DHCP
Inter-VLAN communication works correctly
Network segmentation is implemented
